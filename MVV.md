# vibehawk MVV

> **本ドキュメントはファウンダーが策定した最上位方針であり、いかなるエージェント・プロセスも編集してはならない。**

## Mission
レビューツールに追加課金が要らない世界をつくる。

## Vision
一度の支払いが、つくるすべてに届く世界。

## Values
1. **利用者の契約だけで、完結させる** — 自前のサーバーも追加 API キーも持たず、利用者が既に結んでいる契約の内側で完結する。
2. **観察する、書き換えない** — コードを書き換えず、PR メタデータも触らない。観察し、伝えるところで止まる。
3. **指摘する、強制しない** — severity は付けるが、直すか流すかの裁量は利用者に委ねる。
4. **公式の道を、迂回せず歩く** — 裏 API・スクレイピングを選ばず、公式に案内された経路だけで構築する。
