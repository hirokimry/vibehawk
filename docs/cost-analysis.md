## コスト管理ポリシー

### 予算アラート

- MUST: 月間予算の80%到達時にアラートを発火すること
- MUST: 月間予算の100%到達時に自動停止または承認フローを発動すること

### キャッシュ制御

- MUST: API レスポンスは可能な限りキャッシュし、不要な再呼び出しを防止すること
- MUST NOT: キャッシュ未設定のまま高頻度 API を本番投入しないこと
